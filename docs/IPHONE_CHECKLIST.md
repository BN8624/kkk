# 물리적 iPhone Safari 2.0 검증 체크리스트

검증 대상 URL과 iPhone 모델·iOS 버전을 기록한 뒤 다음을 실제 Safari에서 확인한다.

- [ ] 홈 화면에 설치되고 아이콘·앱 이름이 올바르다.
- [ ] 세로 실행에서 타이틀·설정·HUD 버튼이 잘리지 않는다.
- [ ] 가로 회전 뒤 지도와 오버레이가 다시 맞춰진다.
- [ ] 비행기 모드에서 홈 화면 앱을 종료 후 다시 열 수 있다.
- [ ] 백그라운드 전환 후 진행 중 게임으로 안전하게 복귀한다.
- [ ] 빠른 전투를 시작하고 이동·공격·생산·턴 종료를 수행한다.
- [ ] 캠페인 미션을 시작하고 결과·별·해금이 저장된다.
- [ ] 제작실에서 드래그·핀치·편집·검증·자동 저장을 수행한다.
- [ ] 리플레이를 열고 일시정지·턴 이동·4배속을 수행한다.
- [ ] 영어로 전환하면 즉시 반영되고 재실행 후에도 유지된다.
- [ ] 전체 백업을 다운로드하고 미리보기 후 병합·교체 복구한다.
- [ ] 새 서비스 워커 배포 뒤 게임 중 강제 새로고침 없이 타이틀에 업데이트 안내가 표시된다.

자동 검증 결과와 별개로, 완료한 항목의 화면 녹화 또는 스크린샷과 발견한 기기 전용 문제를 릴리스 기록에 첨부한다.
